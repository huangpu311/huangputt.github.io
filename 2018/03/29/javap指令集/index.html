javap命令

javap定义

javap是java文件的分解器，即是jdk自带的反汇编器，可以查看java编译器为我们生成的字节码。通过它，我们可以对照源代码和字节码，从而了解很多编译器内部的工作。

语法：

javap [命令选项] class ….

Javap 命令用于解析类文件，其输出取决于所用的选项。若没有使用选项，javap将输出传递给他的类public 域及方法，javap将其输出到标准输出设备上。

命令选项：

-help 输出javap的帮助信息

-l 输出行及局部变量表

-b 确保与jdk1.1 javap的向后兼容性

-public 只显示public类及成员

-protected只显示protected和public 类以及成员

-package 只显示包、public和protected类以及成员

-private 显示所有类和成员

-J[flag] 直接将flag传给运行时系统

-s输出内部类型签名

-c输出类中各个方法未解析的代码，即构成java字节码的指令

-verbose 输出堆栈大小、各方法的locals以及args数，和class文件的编译版本

-classpath[路径] 指定javap用来查找类 的路径。如果设置了该选项，则他将覆盖缺省值或CLASSPATH环境变量，目录用分号冒号分割
