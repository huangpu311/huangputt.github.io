####登录到msyql

5.7版本默认对于root帐号有一个随机密码，可以通过 grep “password” /var/log/mysqld.log获得，root@localhost: 此处为随机密码
运行mysql -uroot -p 回车
粘贴随机密码

操作

默认的随机密码是没办法直接对数据库做操作的，需要修改密码，然后，5.7版本用了validate_password密码加强插件，因此在修改密码的时候绝对不是 123456 能糊弄过去的。需要严格按照规范去设置密码
但是，如果想让密码简单点也可以，降低安全策略， 登录到mysql客户端执行如下两条命令

set global validate_password_length=1;

set global validate_password_policy=0;

这样就能设置简单的密码了，但是密码长度必须是大于等于4位

more命令

more命令功能类似于cat，cat 命令是将整个文件的内容从上到下显示在屏幕上。more命令会一页一页的显示,方便使用者阅读，而最基本的执行就是空白键(space)往下一页显示，按b键就会忘回(back)一页显示，而且还有搜寻字符串的功能，more命令从前向后读取文件，因此在启动时就加载整个文件

命令格式 选项 文件

参数 描述
+n 从第n行开始
-n 定义屏幕大小为n行
+/pattern 在每个档案显示前搜寻该字符串，然后从该字符串前两行之后开始显示
-c 从顶部清屏，然后显示
-d 禁用响铃功能
-p 通过清除窗口而不是滚屏来对文件进行幻夜，与-c选项相似
-s 把连续的多个空行显示为一行
-u 把文件内容中的下划线去掉

常用操作

符号 描述
= 输出当前行的行号
q 退出more
空格键 向下滚动已屏
b 返回上一屏

which 命令

which 命令的作用是在path变量指定的路径中搜索可执行文件 所在的位置，它一般用来确认系统中是否安装了指定的软件。

命令格式

which 可执行文件名称

whereis 命令

whereis 命令主要用于定位可执行文件、源代码文件、帮助文件在文件系统中的位置，whereis命令还具有搜索源代码、指定备用搜索路径和索索不寻常的能力。

whereis命令查询速度非常快，这是因为他根本不是在磁盘中漫无目的的乱找，而是在一个数据库中(/var/lib/mlocate/)查询，这个数据库是linux系统自动创建的，包含本地所有文件的信息，并且每天通过自动执行updatedb命令更新一次，这个正是因为这个数据库每天才更新一次，就会使得whereis命令的搜索结果有时候会不准确，比如刚添加的文件可能搜索不到。

命令格式

whereis 选项 文件

常用参数

参数 描述
-b 定位可执行文件
-m 定位帮助文件
-s 定位源代码文件
-u 搜索默认路径下除可执行文件、源代码文件、帮助文件以外的其他文件
-B 指定搜素可执行文件的路径
-M 指定搜素帮助文件的路径
-S 指定搜索源代码文件的路径

locate命令

locate命令跟whereis命令类似，且他们使用的是相同的数据库，但whereis命令只能搜索可执行文件、联机帮助和源代码文件，如果要获得更全面的搜索结果，可以用时locate命令。

locate命令是用来十分复杂的匹配语法，可以使用特殊字元（如*,?）来指定需要查找的样本

命令格式

locate 选项 搜索字符串

常用参数

参数 描述
-q 安静模式，不会显示任何错误信息
-n 至多显示n个输出
-r 使用正则表达式做寻找的条件
-V 显示版本讯息

find 命令

find命令主要作用是沿着文件层次结构向下遍历，匹配符合条件的文件，并执行相应的操作。

命令格式

find 选项 搜索路径 表达式

默认路径是当前目录，默认表达式为-print

表达式可能是由下列成分组成：操作符、选项 、测试表达式以及动作

参数 描述
-print find命令将匹配的文件输出到标准输出
-exec find命令对匹配的文件执行该参数所给出的shell命令
-name 按照文件名来查找
-type 查找某一类型的文件
-prune 使用这一选项可以使find命令不在当前指定的目录中查找
-user 按照文件属性来查询文件
-group 按照文件所属组来查找文件
-mtime -n +n 按照文件的更高时间来查询， -n 表示文件更改时间距现在小于 n天，+n 表示文件更改时间距现在大于 n 天，find 命令还有-atime 和 -ctime 选项
