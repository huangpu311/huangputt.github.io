nginx安装

1、tar -zxvf 安装包

2、.进入文件夹 运行 ./configure –prefix= 目录，默认安装到/usr/local/nginx

3、运行 make & make install

启动命令

进入sbin文件夹 ，运行./nginx即可

./nginx -s stop 停止

./nginx -s quit 退出

./nginx -s reload 重新加载nginx.conf

查看nginx是否运行

ps -ef| grep nginx

验证nginx.conf是否报错

./nginx -t
